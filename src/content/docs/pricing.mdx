Try every Enterprise feature for 30 days — no credit card, no sales call required.

```bash
# Generate a trial license (single-node, all Enterprise features enabled)
curl -X POST http://localhost:8123/admin/license/trial \
  -H "Authorization: Bearer $ADMIN_KEY"
```

## FAQ

**Is ZeptoDB Community really free?**
Yes. The Community edition is free with no license key required. It includes the full core engine, SQL, HTTP API, Python DSL, Arrow Flight, and Parquet HDB — all on a single node. Many teams run it in production.

**What requires an Enterprise license?**
Multi-node clustering, SSO/OIDC, audit log export, advanced RBAC (per-tenant), Kafka connector, migration tools, geo-replication, and rolling upgrades. These features are gated by a signed license key.

**How does license validation work?**
Fully offline — RS256 signature verification with no phone-home. Load via environment variable (`ZEPTODB_LICENSE_KEY`), file (`/etc/zeptodb/license.key`), or HTTP API (`POST /admin/license`).

**What happens when a license expires?**
You get a 7-day warning before expiry, then a 30-day grace period where everything continues working. After the grace period, the instance downgrades to Community edition. No data is ever locked.

**Can I use ZeptoDB in production without Enterprise?**
Absolutely. Many teams run single-node ZeptoDB in production with community support. Enterprise is for teams that need multi-node clustering, SSO, audited deployments, or vendor-backed support.

**Do you offer managed cloud hosting?**
Not yet. It's on our roadmap. For now, see our [Production Deployment](/deployment/production_deployment/) and [Kubernetes Operations](/operations/kubernetes_operations/) guides — or [talk to us](/contact/) about a bespoke deployment.
