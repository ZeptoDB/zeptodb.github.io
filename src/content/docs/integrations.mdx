## Ingestion

<CardGrid>
  <Card title="Kafka" icon="star">
    Built-in Kafka consumer with batch ingestion. Up to 3.2M events/sec from Kafka topics. The lingua franca for enterprise streaming.

    [Feed Handler Guide →](/feeds/feed_handler_guide/)

    > Kafka connector requires Enterprise license.
  </Card>
  <Card title="FIX 4.4" icon="star">
    Native FIX parser for market-data and order feeds. 1.1M messages/sec with 420ns parse latency.

    [Feed Handlers →](/feeds/feed_handler_complete/)
  </Card>
  <Card title="ITCH" icon="star">
    NASDAQ ITCH handler for L3 order-book data. Lossless, nanosecond-timestamped.

    [Feed Handlers →](/feeds/feed_handler_complete/)
  </Card>
  <Card title="Binance WebSocket" icon="star">
    Real-time crypto market data ingestion from Binance exchange.

    [Feed Handlers →](/feeds/feed_handler_complete/)
  </Card>
</CardGrid>
