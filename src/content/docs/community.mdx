## Contributing

We welcome contributions of all kinds:

- Bug reports and feature requests → [GitHub Issues](https://github.com/zeptodb/zeptodb/issues)
- Code contributions → [Contributing Guide](https://github.com/zeptodb/zeptodb/blob/main/CONTRIBUTING.md)
- Documentation improvements → [Docs repo](https://github.com/zeptodb/zeptodb/tree/main/docs)
- Benchmarks and performance reports
- Integration examples and tutorials

### Getting Started

```bash
git clone https://github.com/zeptodb/zeptodb.git
cd zeptodb
mkdir -p build && cd build
cmake .. -G Ninja -DCMAKE_BUILD_TYPE=Debug \
  -DCMAKE_C_COMPILER=clang-19 -DCMAKE_CXX_COMPILER=clang++-19
ninja -j$(nproc)
ninja zepto_tests && ./tests/zepto_tests
```

## Roadmap

See the [public roadmap](https://github.com/zeptodb/zeptodb/milestones) for upcoming features and priorities.

### Near-Term
- Web UI / Admin Console
- JDBC/ODBC drivers
- Managed cloud offering

### Medium-Term
- MQTT / OPC-UA connectors for IoT
- ROS2 plugin for robotics
- Geo-replication
- Arrow Flight improvements

### Long-Term
- DSL AOT compilation (Nuitka/Cython)
- ClickHouse wire protocol compatibility
- Cloud marketplace listings (AWS, GCP)

Have a feature request? [Open a discussion](https://github.com/zeptodb/zeptodb/discussions).
